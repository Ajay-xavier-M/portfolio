

Portfolio made with react.js. it has my projects & skills among others.

## Tech Stack

**Technology:** React.js, CSS3, Html5

## Run Locally

Clone the project

```bash
  git clone https://github.com/Ajay-xavier-M/portfolio-1.git
```

Go to the project directory

```bash
  cd portfolio-1
```

Run

```bash
  npm install
  yarn start
```
